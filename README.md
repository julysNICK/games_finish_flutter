## Starting

  Follow the instructions below so that you can configure the project locally.
  
###  Installation

1. Clone the repo
   ```sh
   git clone https://github.com/julysNICK/games_finish
   ```
2. Install go packages
   ```sh
   flutter pub get
   ```

3. Run the project
   ```sh
   flutter run
   ```
